# Festigo - Campus Event Registration Portal
Festigo is a modern, interactive **campus event management web application** that allows students to explore events, register for them, and manage their participation seamlessly. Built using **HTML, CSS, and JavaScript**, it focuses on simplicity, responsiveness, and a smooth user experience.

# Features

## Authentication System
- User Sign Up with validation (name, email, password, phone)
- Login system with localStorage-based persistence
- Logout functionality
- Auto session restore on page reload

## Event Exploration
- Browse multiple campus events across categories:
  - Tech
  - Cultural
  - Creative
- Clean card-based UI for events
- Event details including time, date, and team size

## Event Registration
- Register for events after login
- Prevent duplicate registrations
- View all registered events in a dedicated tab

## Smart Filtering System
- Search events by name or description
- Filter by:
  - Category (Tech / Cultural / Creative)
  - Team size
  - Time (Morning / Afternoon / Evening)
  - Date range (Start & End date)

## User Experience
- Responsive and modern UI design
- Landing page with feature highlights
- Interactive overlays for login, signup, and event registration
- Smooth transitions and hover effects

## Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & layout (Grid, Flexbox, animations)
- **JavaScript** – Logic, validation, filtering, and state management
- **LocalStorage** – Persistent user and registration data
